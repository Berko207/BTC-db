# Contributing to BTC-db

Thank you for your interest in improving this Bitcoin historical context database!

## Guidelines

- All additions must be **factually accurate** and sourced from reputable public records.
- Prefer primary sources (Bitcoin whitepaper, official announcements, blockchain data).
- For price data, use well-known references and note approximations.
- Keep tables in `MILESTONES.md` parseable (simple markdown tables).
- Narrative history goes in `HISTORY.md`.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request with clear description

## What We're Looking For

- Additional protocol upgrade details
- More granular on-chain milestones
- Regulatory events by jurisdiction
- Improved data tables for trading/analytics use cases
- Corrections to existing entries

---

*This repo aims to be a trusted, neutral reference for Bitcoin context.*
